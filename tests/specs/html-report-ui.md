# HTML Report UI

Verify the HTML report displays correctly with the new improved UI design.

## Test Scenario
Open the most recent HTML report and verify all UI elements are present and functional.

## Expected Visual Elements

**Hero Section:**
- Large status indicator (PASSED/FAILED/NEEDS REVIEW) with colored border
- Status icon and message clearly visible
- Centered layout with prominent typography

**Filter/Summary Section:**
- Five clickable summary boxes: All Tests, Passed, Needs Review, Failed, Errored
- Each box shows count and label
- "All Tests" box is blue and active by default
- Boxes have distinct colors and hover effects

**Search Bar:**
- Full-width search input below summary boxes
- Clear placeholder text

**Test Results:**
- Test cards with colored left borders
- Expandable details on click
- Test names and durations visible
